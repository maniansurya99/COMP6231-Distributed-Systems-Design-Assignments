1) Run the serverMain.java

2) Run the clientMain.java

3) The first client's name should be entered

4) Run the client for multiple times to get multiple clients connected to the server

5) Follow steps 3 and 4 iteratively

6) In the console window, use the "Display Selected Console", on the top-right corner to choose which client needs to enter the text in the console first

7) Follow step 6 to allow multiple clients to chat at any order

8) All the inputs entered by all the clients are visible to every other client, and not just the server

9) Screenshots are attached in the project folder.
